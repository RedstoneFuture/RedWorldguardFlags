# RedWorldguardFlags
Extra Flags used on the redstoneworld.de server.

## Flags:

| Flag                     | Description                                                  | Default State |
|--------------------------|--------------------------------------------------------------|---------------|
| `dispense-nbt-spawneggs` | Toggles if spawn-eggs with custom NBT data will be dispensed |  true         |
